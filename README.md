\# Zybo LED Blinking



A basic FPGA project implemented in Verilog HDL for the Zybo Z7 board.



\## Files

\- led\_blinking.v

\- led\_blinking\_constraints.xdc



\## Tools Used

\- Vivado

\- Verilog HDL



\## Description

This project demonstrates LED blinking on the Zybo FPGA board using a clock divider.
